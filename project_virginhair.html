Интернет-магазина
«Virgin hair»

Цель

Создать продающий интернет-магазин косметики для волос, чтобы перенаправить трафик из соцсетей на сайт и увеличить конверсию в покупку благодаря прогреву аудитории. Главная страница должна четко описывать преимущества продукта.

Решение

Интернет-магазин косметики сделан на wordpress для удобного управления новыми линейками продукции, каталогом и ассортиментом, который постоянно расширяется. Главная страница разработана с нуля на html, css, js для гибкого и быстрого изменения дизайна под требования клиента. Реализован функционал для изменения текстов, баннеров и ведения блога. Настроен эквайринг, условия доставки и проработана корзина с оформлением заказа.
